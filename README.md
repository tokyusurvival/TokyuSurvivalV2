## 简介

近日，东京某理工科高职院校的学生制作了一款以东京疫情封控为背景的校园生存文字游戏《东急求生记》……

东京急行大学简称东急，罗马字Tokyu，拼音`Dongji`。玩家将在三个月校园封控的重重挑战下，保护自己的健康与脆弱的心灵，探索玄妙的校园社会百态。譬如：

+ 有限条件下的美食烹饪
+ 克苏鲁文化
+ 《开端》般的无尽循环
+ 修行未来的社畜宝典
+ 与各种势力作斗争

参与内测的玩家反馈道，整个游戏挑战十足，玩家在游戏中屡屡破防，部分玩家首次试玩仅获得了生存三天的好成绩。
